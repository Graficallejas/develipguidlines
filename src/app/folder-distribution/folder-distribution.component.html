Estructurar carpetas es el arte de organizar nuestro proyecto, Angular cli nos provee una estructura básica de la cual podemos partir para iniciar nuestro proyecto pero se puede volver algo tedioso a medida que nuestro proyecto empieza a tener muchos módulos.

Para hacer más escalable el proyecto se ha decidido seguir el siguiente patrón de estructura de carpetas:
Partiendo de la carpeta base (app) vamos a encontrar la carpeta modules donde tendremos los distintos módulos del sistema, cada uno de estos trabajando de forma independiente con carga perezosa . Cada módulo en sí contiene los distintos componentes principales que lo conforman y estos su manejador de estado, servicios, archivo de interfaces y componentes complementarios o modales que este mismo requiera.
