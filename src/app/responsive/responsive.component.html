Se recomienda que para el diseño de plataformas se utilice la metodología Mobile first.

Para el diseño receptivo se sugiere utilizar los break points delimitados en el siguiente enlace https://material.io/archive/guidelines/layout/responsive-ui.html#

  • Movil: entre 320 y 479 pixeles
  • Tablet pequeña: entre 480 y 600 píxeles
  • Tablet grande: entre 840 y 1280 píxeles
  • Laptop: entre 1280 y 1440 píxeles
  • Pantallas grandes: más de 1440 píxeles
Buenas prácticas para generar un Media Queries

El diseño responsivo tiene como objetivo adaptar la visualización de una página web a cada dispositivo de la mejor manera posible, recomendamos tener en cuenta esto basándonos en la sugerencia de https://www.w3.org/TR/mediaqueries-3/
